Análisis de gastos médicos. Data set "Insurance". 

Descripción.
Este proyecto utiliza el dataset clásico insurance.csv, que contiene información demográfica y estilo de vida de pacientes junto con sus gastos médicos anuales.
El objetivo es aplicar Regresióon Lineal para identificar qué variables influyen más  en los costos.

Contenido.
- "insurance.csv." : dataset con los registros de pacientes.
- "gastos_medicos.ipynb" : notebook con el análisis exploratorio y el modelo de regresión lineal.
- "requirements.txt" : requerimientos de las librerías utilizadas en el trabajo.
- "readme.txt" : este archivo con la explicación del proyecto.

Metodología.
1- Exploración de datos: análisis descriptivo y visualización de las variables.
2- Prepocesamiento: transformación de variables categóricas a dummies.
3- Modelo de Machine Learning: ajuste del modelo de Regresión Lineal y evaluación.

Principales hallazgos.
- La variable "smoker" es la más determinante: ser fumador incrementa significativamente los gastos.
- "BMI" y "age" también muestran una relación directa con los gastos.
- El resto de las variables tienen menor impacto.

Tecnologías utilizadas.
- Python 3
- Pandas
- Scikit-learn
- Matplotlib
- Numpy
- Seaborn

Conclusión.
El modelo confirma que los hábitos de fumador y factores generales de salud (como la masa corporal) son claves para explicar los gastos médicos.
Este análisis puede servir como base para estudios mas avanzados de regularización o machine learning ya que este trabajo en su totalidad esta realizado con fines prácticos y educativos.
